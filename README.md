# GUII_II_FINAL_PROJECT-ConnectME
Contributors: Joseph Calles, Tammy Liu, Benjamin Friedman

http://weblab.cs.uml.edu/~jcalles/ConnectME

Final project for GUI II (Graphical User Interface Programming II). ConnectME is a full stack project dedicated to providing mentorship opportunities for computer science students by connecting them to real world software professionals. Students can create an account to seek out mentors in their desired field of interest (full stack development, systems programming, AI, machine learning etc.), and real world computer science professionals interested in mentoring can create an account to help out students.

Languags and Technology:
- HTML5
- CSS and Bootstrap 5
- JavaScript and jQuery
- PHP and MySQL

NOTE: we have implemented email verification on weblab. Please use your own emails when grading so you can see how the email verification works.
